## Installation

### 1. Créer un environnement virtuel

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

```bash
pip install pandas nltk gensim scikit-learn transformers torch
```
### 3. Telecharger les ressources nltk
```bash
python download_nltk.py
```
